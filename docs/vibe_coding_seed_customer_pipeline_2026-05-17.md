# TraceMind Vibe Coding Seed Customer Pipeline - Batch A0

Date: 2026-05-17

Goal: 找到更小、更早期、更接近 TraceMind 核心客户的种子用户：非技术/弱技术创始人，靠 vibe coding 工具把产品做上线，但还不知道真实用户怎么用。

## Revised ICP

这批优先找：

- 个人或 2-3 人团队。
- 明确使用 Lovable、Base44、Floot、Bolt、Replit、Cursor、Claude Code 等工具做出产品。
- 产品已经上线、正在找反馈、waitlist、前 10 个用户、Product Hunt 支持，或刚接入支付。
- 不像成熟技术团队那样已经有完善 analytics。
- 他们的问题不是“怎么写代码”，而是“产品上线后用户到底卡在哪”。

暂时降级：

- MCP/devtool 基础设施团队：适合生态合作，但不是首批最贴近的试点客户。
- 成熟 SaaS 团队：可能需要更长销售周期。
- 纯 demo、没有公开产品链接或没有用户目标的项目。
